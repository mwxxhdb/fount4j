# Fount4j

基于 SpringCloud 的后台管理系统框架。

本框架还在开发中，欢迎持续关注。

## 技术栈

- Spring Boot 2.4.5
- Lombok
- Thymeleaf
- [Fomantic-UI](https://fomantic-ui.com/)
- 认证和授权 [sa-token](https://github.com/dromara/sa-token)
- Redis
- MongoDB
- MySQL
- H2
- Spring Data JPA

## 和其他常见框架的区别

- 没有前后端分离
  - 干掉 node_modules !!!
  - 干掉 npm !!!
- 没有使用 Shiro 或 Spring Security 框架
- Service 在线则显示对应的菜单，不在线则不显示
